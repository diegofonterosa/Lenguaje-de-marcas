Conceptos Clave de HTML

<h1> a <h6> (Encabezados)
Las etiquetas de encabezado se usan para definir títulos o subtítulos en una página web.
  • <h1> es el título principal (más importante).
  • <h6> es el título menos importante.
Más información: Encabezados en MDN

Ejemplo de uso: 
<h1>Bienvenidos a mi página</h1>
<p> (Párrafo)
La etiqueta <p> se usa para escribir párrafos de texto.

Características:

  • Separa bloques de texto.
  • Es fácil de usar y entender.
Aprende más en: Párrafos en MDN

Ejemplo de código: 
<p>Este es un párrafo de ejemplo.</p>
<a> (Enlace)
Sirve para crear enlaces a otras páginas o sitios web.
  1. Puedes poner enlaces internos (dentro de tu página).
  2. También enlaces externos (otras páginas en Internet).
Más información aquí: Enlaces en MDN

Ejemplo de uso: 
<a href="https://www.google.com" target="_blank">Ir a Google</a>
<img> (Imagen)
Se utiliza para insertar imágenes en la página web.
  • Debe tener el atributo src (la ruta de la imagen).
  • Es recomendable usar el atributo alt para describir la imagen.
Lee más aquí: Imágenes en MDN

Ejemplo de código: 
<img src="foto.jpg" alt="Foto de paisaje">
<ul> y <ol> (Listas)
Se usan para crear listas:
  • <ul> es una lista desordenada (con viñetas).
  • <ol> es una lista ordenada (con números).
Más detalles aquí: Listas en MDN

Ejemplo de listas: 
<ul>
    <li>Manzana</li>
    <li>Banana</li>
    <li>Naranja</li>
</ul>
